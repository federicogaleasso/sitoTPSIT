Cos'è il JSON?

JSON ( JavaScript Object Notation ) è un semplice formato per lo scambio di dati. Per le persone è facile da leggere e scrivere, mentre per le macchine risulta facile da generare e analizzarne la sintassi. Si basa su un sottoinsieme del Linguaggio di Programmazione JavaScript, Standard ECMA-262 Terza Edizione. JSON è un formato di testo completamente indipendente dal linguaggio di programmazione, ma utilizza convenzioni conosciute dai programmatori di linguaggi della famiglia del C, come C, C++, C#, Java, JavaScript, Perl, Python, e molti altri. Questa caratteristica fa di JSON un linguaggio ideale per lo scambio di dati.

JSON è basato su due strutture:

  • Un insieme di coppie chiave:valore
  • Un elenco ordinato di valori

    • Di seguito è presente un esempio di JSON: 

    {
  "nome":"Mario",
  "cognome":"Rossi",
  "eta":"20"
}

    Oggetto


    Un oggetto è una serie non ordinata di chiave:valore. Un oggetto inizia con ' { ' e finisce con ' } '. Ogni nome è seguito da ' : ' e le coppie di chiave:valore sono separate da ' , '.





    Vettore


    Un vettore è una raccolta ordinata di valori. Comincia con ' [ ' e finisce con ' ] '. I valori sono separati da ' , '.





    Valore


    Un valore può essere una stringa tra virgolette, un numero, true, false, null, un oggetto o un array. Queste strutture possono essere annidate.





    Stringa


    Una stringa è una raccolta di zero o più caratteri Unicode, tra virgolette; per le sequenze di escape utilizza la barra rovesciata. Un singolo carattere è rappresentato come una stringa di caratteri di lunghezza uno.





    Numero


    Un numero è molto simile ad un numero C o Java, a parte il fatto che i formati ottali e esadecimali non sono utilizzati.





    Caratteri di spaziatura


    I caratteri di spaziatura possono essere inseriti in mezzo a qualsiasi coppia di token.








    Parsing di un file JSON


    Ci sono diversi metodi per eseguire il parsing di un file JSON: il più comune ed il più utilizzato è l'utilizzo della funzione fetch().
    La funzione fetch(), simile ad una promise, è una funzione ASINCRONA ( il codice viene eseguito in parallelo con le altre operazioni ). La fetch() scarica tutto il contenuto della pagina web ( HTML, CSS, JSON, ecc... ), per questo richiede un parametro: l'URL.

    La fetch() utilizza due funzioni chiamate .then():

  • Il primo .then() viene richiamato dopo lo scaricamento di tutti i dati della pagina e all'interno è presente la variabile che contiene tutti i dati scaricati come testo, che verranno poi trasformati in JSON.
  • Il secondo .then() viene richiamato dopo l'esecuzione del primo .then() e serve a prendere i dati che verranno visualizzati nella pagina.


    • const URL = "your_url"
fetch(URL)
.then(response => response.json())
.then(data => {
  console.log(data)
})
