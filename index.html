Impara i concetti di JSON, XML, JavaScript Avanzato, React e API

JSON

JSON ( JavaScript Object Notation ) è un semplice formato per lo scambio di dati.

XML

XML ( Extensible Markup Language ) è un linguaggio che consente la rappresentazione di documenti.

JS Avanzato

JavaScript Avanzato comprende le funzionalità più avanzate di JavaScript.

React

React è una libreria JavaScript che permette di creare le SPA ( Single Page Application )

API

API ( Application Programming Interface ) consentono a software e applicazioni di comunicare tra loro.
