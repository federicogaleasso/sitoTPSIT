Cos'è JavaScript Avanzato?


JavaScript è un linguaggio di programmazione che consente di aggiungere funzionalità interattive e altri contenuti dinamici alle pagine web. JavaScript Avanzato si riferisce alle tecniche di programmazione e alle funzionalità più avanzate del linguaggio, come ad esempio le Promises.

    Template String


    Le template string consentono di creare stringhe più leggibili rispetto alle stringhe tradizionali. Per definire una template string, utilizziamo le virgolette inverse (` `), anziché le virgolette singole o doppie. Inoltre, all'interno di una template string, è possibile includere espressioni JavaScript utilizzando la sintassi ${ ... }.


    Di seguito è presente un esempio di utilizzo di template string: 

    const stampa=()=>{
    let nome = "mario";
    let cognome = "rossi";

    console.log("Benvenuto " + nome + " " + cognome);

    //TEMPLATE STRING
    console.log(`Benvenuto ${nome} ${cognome}`);
}



    Funzione map()


    La funzione map() è un metodo degli array che consente di iterare su tutti gli elementi dell'array ed applicare una trasformazione a ciascun elemento grazie alla funzione callback ( che ha come parametri l'elemento e l'indice ). Restituisce un nuovo array contenente le eventuali modifiche.


    Di seguito è presente un esempio di utilizzo della funzione map(): 

    const stampa=()=>{
  let array=[5,4,6,1,2,9]

  console.log(
      //FUNZIONE MAP
      array.map((elemento,indice)=>{
          return elemento*2;  //Moltiplica *2 ogni elemento dell'array
      })
  )
}
