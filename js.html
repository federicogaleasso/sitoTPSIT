Cos'è JavaScript Avanzato?


JavaScript è un linguaggio di programmazione che consente di aggiungere funzionalità interattive e altri contenuti dinamici alle pagine web. JavaScript Avanzato si riferisce alle tecniche di programmazione e alle funzionalità più avanzate del linguaggio, come ad esempio le Promises.

    Template String


    Le template string consentono di creare stringhe più leggibili rispetto alle stringhe tradizionali. Per definire una template string, utilizziamo le virgolette inverse (` `), anziché le virgolette singole o doppie. Inoltre, all'interno di una template string, è possibile includere espressioni JavaScript utilizzando la sintassi ${ ... }.


    Di seguito è presente un esempio di utilizzo di template string: 

    const visualizza=()=>{
    let nome = "mario";
    let cognome = "rossi";

    console.log("Benvenuto " + nome + " " + cognome);

    //TEMPLATE STRING
    console.log(`Benvenuto ${nome} ${cognome}`);
}

    Funzione map()


    La funzione map() è un metodo degli array che consente di iterare su tutti gli elementi dell'array ed applicare una trasformazione a ciascun elemento grazie alla funzione callback ( che ha come parametri l'elemento e l'indice ). Restituisce un nuovo array contenente le eventuali modifiche.


    Di seguito è presente un esempio di utilizzo della funzione map(): 

    const visualizza=()=>{
  let array=[5,4,6,1,2,9]

  console.log(
      //FUNZIONE MAP
      array.map((elemento,indice)=>{
          return elemento*2;  //Moltiplica *2 ogni elemento dell'array
      })
  )
}

    Promise


    La promise è una funzione asincrona in grado di gestire il flusso asincrono di un programma in modo più efficente. Viene utilizzata specialmente per le operazioni che richiedono molto tempo come le chiamate a API o l'accesso a database, in modo tale da non bloccare l'esecuzione del codice.


    Di seguito è presente un esempio di utilizzo della promise: 

    const visualizza=()=>{

  //Funzione ASINCRONA
  const primaFunzione=async()=>{
      sleep(3000)
      console.log("Io sono la prima funzione")
      return "Scaricato"
  }

  //Funzione SINCRONA
  const secondaFunzione=()=>{
      console.log("Io sono la seconda funzione")
  }

  primaFunzione()

  //PROMISE
  .then((parametro)=>{
      //elaborazione...
  })

  secondaFunzione();
}

    Destrutturazione


    La destrutturazione consente di estrarre valori da oggetti o array in modo veloce e intuitivo. Consente di scomporre una struttura complessa in parti più semplici, assegnando i valori estratti a variabili separate.


    Di seguito è presente un esempio di utilizzo della destrutturazione: 

    const visualizza =()=>{

  let persona={
      nome:"Mario",
      cognome:"Rossi",
      eta:25,
      sesso:"M",
      telefono:{
          cellulare:3398576791,
          fisso:3479894532
      }
  }

  //Prima della destrutturazione
  console.log(persona.nome + " " + persona.cognome + " " + persona.telefono.cellulare)
  
  //DESTRUTTURAZIONE: estrapoliamo in questo caso il nome, il cognome e il cellulare dall'oggetto persona. In questo modo evitiamo di scrivere persona.nome ecc...
  const { nome, cognome, telefono:{cellulare} } = persona

  //Dopo la destrutturazione
  console.log(nome + " " + cognome + " " + cellulare)
}

    Operatore Spread


    L'operatore spread serve per fare una copia di un array. La sintassi dell'operatore spread è rappresentata da tre puntini consecutivi ( ... ).


    Di seguito è presente un esempio di utilizzo dell'operatore spread: 

    const visualizza =()=>{

  let array_a = ['pizza', 'sushi', 'pasta']

  //OPERATORE SPREAD
  let array_b = [...array_a]

  console.log(array_b)
}
